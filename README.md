# PokemonDataCenter

> Pokemon data server


## How to use

> get whole pokemons on first generation

```url
/**
 * @return {array} pokemons
 */
http://data.readymade.or.kr/pokemon
```

> get single pokemon on first generation by number  


```url
/**
 * @params {number} [no]
 * @return {array} pokemon
 */
http://data.readymade.or.kr/pokemon?no=1
```
